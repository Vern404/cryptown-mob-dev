# cryptown-mob-dev
 A simple mobile application for cryptocurrencies portfolio that contains crypto news, top 100 crypto and its details
 
# Cryptown features:
 1. Sign In
 2. Sign Up
 3. Crypto List
 4. Crypto Details
 5. Crypto Line Chart
 6. Exchange list
 7. News
 8. Forum
 9. User profile
